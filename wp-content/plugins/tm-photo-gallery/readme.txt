== Description ==

Showcase your project images in elegant, dynamic and easy-to-navigate galleries. With this WP plugin, you can create a gallery with a single click and add it to any post or page of your site.


TM Photo Gallery is a versatile solution for any website type, whether you run a blog, personal portfolio or online store. It has multiple settings to configure galleries according to your needs.


__Photo Editor.__ This feature allows you to edit images in different ways. For example, you can rotate a photo clockwise or counterclockwise and set a focal point on the necessary area.


__Gallery Types.__ Choose the type of the gallery layout from grid, masonry and justify options. You can also set the number of columns and a cover photo for your gallery with a single click.

__Animation Effects.__ Spice your galleries up with smooth animation effects such as fade, scale, flip, helix, and much more.


__Lightbox.__ For an easy viewing experience, visitors can take a closer look at every image in a lightbox. You can optionally enable an autoplay option and full-screen mode, use thumbnails, and display nav arrows.


__Navigation.__ Provide easier navigation for viewers by means of filters and pagination options. Here’s how you can configure them:

- set the type of filter display and criteria to filter images by;
- specify the number of images per page;
- add “Load More” button, “Load More” grid, and pagination block.


__Meta Settings.__ TM Photo Gallery also comes with meta settings to use for the following purposes:


- specify labels for sets and albums;
- add icons, titles, and descriptions;
- set a custom number of words for descriptions;
- enable a meta counter.


If you want to create engaging galleries and easily customize them, TM Photo Gallery is worth a try.

== Installation ==

1. Upload "TM Gallery" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
